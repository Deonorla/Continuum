WHAT STREAM ENGINE DOES
Stream Engine is a Web3 payment infrastructure platform built on Polkadot that enables AI agents to autonomously pay for API services using streaming micropayments in DOT/Polkadot tokens. Key capabilities:

x402 HTTP Standard: Agents hit an API → server returns 402 → x402 manifest describes payment options → agent pays and retries automatically
FlowPayStream Smart Contract: Polkadot tokens locked in contract, streamed per-second to recipients
Gemini AI Decision Engine: Decides whether to pay per-request (single DOT transfer) or initiate a continuous stream
Multi-agent coordination: Multiple AI agents orchestrating service calls across a network
Human oversight: Kill switches, rate limiters, safety controls
Aptos RWA Module: DeFi protocol tokenizing real estate, vehicles, commodities as NFTs → yields streamed on-chain → KYC/AML built-in → Flash Advance (borrow against future yield)
